pub struct LocalKeyPair(_);

Key pair that is implemented by signing using a private key.

This implements the KeyPair trait.

Obviously, given the nature of this kind of signer, the private key is contained in the memory space of the process, and thus can be leaked with any sort of memory disclosure vulernability (not necressarily in the TLS implementation).
