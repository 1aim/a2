Error in parsing certificate.

Variants

Toplevel(CertificateToplevelError)

Error in reading toplevel.

This error occurs if anything goes wrong in reading the top level of the certificate, which includes the topmost SEQUENCE and the fields to-be-signed data, signature algorithm and signature elements within that sequence.

The parameter is structure giving more detailed reading error.

Version(CertificateVersionError)

Error in parsing certificate version.

This error occurs if anything goes wrong in parsing the certificate version, or if the certificate version is not supported.

The parameter is structure giving more detailed parsing error.

ExtensionEntry(usize, CertificateExtensionError)

Error in parsing certificate extension.

This error occurs if extension structure can not be parsed.

  • The 1st parameter is number of extension (starting from 0) that triggered the parse error.
  • The 2nd parameter is more detailed parsing errorr.
