Error parsing certificate extension.

Variants

Toplevel(Asn1Error)

Error reading the extension toplevel.

This error occurs if the topmost SEQUENCE in the extension can not be read.

The parameter is structure giving more detailed reading error.

Oid(Asn1Error)

Error reading the extension OID.

This error occurs if the OID of the extension can not be read.

The parameter is structure giving more detailed reading error.

Error reading the extension critical flag

This error occurs if the critical flag of extension can not be read.

  • The 1st parameter is string representation of the OID of the extension.
  • The 2nd parameter is more detailed reading error.
