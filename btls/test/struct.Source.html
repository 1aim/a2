pub struct Source<'a>(_, _);

Source that provodes data out of byte slice.

This structure reads byte slice as various data types, in order. Handy for parsing all sorts of protocols.

fn read_asn1_value<Tag: Asn1TagMatch, E, EM>(&mut self,
                                             xtag: Tag,
                                             maperr: EM)
                                             -> Result<Asn1Tlv<'a>, E> where EM: Fn(Asn1Error) -> E

Read an ASN.1 DER TLV with known type out of source.

Only lengths up to 16777215 are read, larger lengths trigger an error.

Parameters:

  • self: The source to read from.
  • xtag: The expected tag (AnyAsn1Tag if any tag is to be allowed).
  • maperr: Closure to map the error on failure.

Return value:

  • The newly-read ASN.1 TLV.

Failures:

  • If the source does not have enough data to satisfy request, fails with Asn1Error::Truncated.
  • If the input is not canonical, fails with Asn1Error::NotDer.
  • If the length is greater than 16777215, fails with Asn1Error::LengthTooBig.
  • If the tag is greater than 263-1, fails with Asn1Error::TagTooBig.
  • If the tag is unexpected, fails with Asn1Error::UnexpectedType.
