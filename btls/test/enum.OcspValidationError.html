Error in validating OCSP response.

Variants

SCertParse(CertificateError)

Error in parsing subject certificate.

This error occurs if parsing of the subject certificate fails.

The parameter is more detailed parsing error.

RCertParse(CertificateError)

Error in parsing responder certificate.

This error occurs if parsing of the responder certificate fails.

The parameter is more detailed parsing error.

BasicResponse(Asn1Error)

Error in reading BasicResponse

This error occurs if reading of the BasicResponse fails.

The parameter is more detailed reading error.

ResponseSignatureError

OCSP response signature does not verify.

This error occurs if the signature in OCSP response does not verify.

ResponderSignatureError

OCPS responder certificate signature does not verify.

This error occurs if the signature in OCSP responder certificate does not verify.

ResponderNotYetValid(i64)

Responder certificate is not yet valid.

This error occurs if the responder certificate sent is not yet valid.

The parameter is the timestamp when the validity of the responder certificate would begin.

ResponderExpired(i64)

Responder certificate has expired.

This error occurs if the responder certificate sent has expired.

The parameter is the timestamp when the validity of the responder certificate ended.

ResponderNotValidForUse

Responder certificate not valid EKU for OCSP responder.

This error occurs if the responder certificate does not have valid ExtendedKeyUsage for OCSP.

ResponderNotIssuedByParent

Responder certificate not issued by the issuer certificate.

This error occurs if the responder certificate is not issued by the same issuer as the certificate being checked.

MultipleResponderCertificates

Multiple responder certificates were sent.

This error occurs if OCSP response contains multiple responder certificates.
