Extension trait implementing non-panicing subslicing

This trait provodes method to extract subslice from slice without panicing. Great for extracting slices with indexes of unknown provenance.

This trait is blanket-implemented for all slice types ([T]).

Required Methods

fn slice_np<X: RangeTrait>(&self, r: X) -> Option<&Self>

Extract subslice.

Parameters:

  • self: The slice to subslice.
  • r: The range. Can be any of the four usual slice range types.

Return value:

  • If slicing was successful, Some(x), where x is the slice.
  • If the range is out of bounds, None.

fn split_at_np<'a>(&'a self, pt: usize) -> Option<(&'a Self, &'a Self)>

Split a slice into two.

Parameters:

  • self: The slice to subslice.
  • pt: The point to slice at.

Return value:

  • If slice can be split at given point, Some((a, b)), where a and b are the slices.
  • If the point is out of range, None.
